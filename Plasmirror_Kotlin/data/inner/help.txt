Программа для расчета спектров отражения, пропускания и поглощения света в слоистых наноструктурах методом матриц переноса.
В случае, когда в описании структуры присутствует только один слой, возможен расчет его диэлектрической проницаемости и
коэффициента преломления.

Типы слоёв:
1 - GaAs
2 - AlGaAs
3 - слой с постоянным коэффициентом преломления (n; k)
4 - GaAs с экситонами
5 - AlGaAs с экситонами
6 - слой с постоянным коэффициентом преломления (n; k) с экситонами
7 - слой AlGaAs, содержащий неупорядоченную систему нановключений металла с указанными параметрами в объёме AlGaAs,
    рассматриваемый в модели эффективной среды Максвелла-Гарнетта
8 - одиночный слой нановключений металла в AlGaAs (Persson model (Phys. Rev. B, 28, PP. 4247 (1983)))

Описания слоёв группируются в периоды.
Каждый период предваряется строкой вида: "xN", где N - количество повторений периода (N >= 1)

Форма записи типов 1, 2, 4, 5, 7, 8 должна соответствовать формату:
"тип-i", где i указывает способ расчета диэлектрической функции матрицы AlGaAs:
1 - по модели [1]
2 - по модели [2]
3 - по модели [2], но при энергиях ниже E0 мнимая часть диэлектрической проницаемости положена равной нулю
(было раньше) 3 - по модели [1] до точки пересечения (ниже Eg) с [2] и по модели [2] выше точки пересечения.


Параметры:
1, d
2, d, k, x
3, d, n
4, d, w0, gamma0, gamma
5, d, k, x, w0, gamma0, gamma
6, d, n, w0, gamma0, gamma
7, d, k, x, w_plasma, gamma_plasma, f, eps_inf
8, d, k, x, lattice_factor, w_plasma, gamma_plasma, eps_inf (d = 2 * R_particle)

d                    - толщина, нм
n                    - комплексный показатель преломления в форме (a; b)
k                    - мнимая часть n
x                    - концентрация AlAs, [0; 1]
w0                   - резонсная частота экситона, эВ
gamma0               - радиационное затухание экситона, эВ
gamma                - нерадиационное затухание экситона, эВ
w_plasma             - плазменная частота для нановключения металла в вакууме, эВ
gamma_plasma         - константа затухания плазмона, эВ. gamma_plasma = (h_planck/2Pi) / tau, tau ~ 3E-15 сек, h_planck = 6.582E-16 эВ * сек
f                    - объемная доля нановключений в AlGaAs, [0; 1]
lattice_factor       - масштабирующий коэффициент, связывающий эффективный период квадратной решетки нановключений с их радиусом


Все параметры слоёв необходимо именовать, используя соответствущий набор ключевых слов (например, "type"),
знак "=" после имени параметра и перед его значением обязателен.
Описание слоя не обязательно должно располагаться на одной строке.

Пример:
x1                                        - период
type = 1-2, d = 5                         - слой GaAs
type = 2-2, d = 130, k = 0.0, x = 0.31    - слой AlGaAs

x24
type = 8-2, d = 20, k = 0.0, x = 0.31,    - описание слоя в несколько строк
lattice_factor = 8.0,
w_plasma = 7.38, gamma_plasma = 0.2,
eps_inf = 1.0

// type = 2-2, d = 112, k = 0.0, x = 0.31 - однострочный комментарий

/*                                        - многострочный комментарий
x24
2-1, 12.15, 0.005, 0.0
*/
