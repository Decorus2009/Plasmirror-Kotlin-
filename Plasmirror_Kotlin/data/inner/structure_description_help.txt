Типы слоёв:
1 - GaAs без экситонов
2 - AlGaAs без экситонов
3 - произвольный слой с постоянным n слой без экситонов
4 - GaAs с экситонами
5 - AlGaAs с экситонами
6 - произвольный с постоянным n слой с экситонами
7 - метаматериал AlGaAs-(AsSb) в модели эффективной среды Максвелла-Гарнетта
8 - слой наночастиц металла в AlGaAs (рассматривается двумерная периодическая решетка НЧ, см. статью Persson)

Группу слоёв в периоде следует предварить строкой вида "xN", где N - количество периодов, [1; inf)

Формат описания слоя:
1, d
2, d, k, x
3, d, n
4, d, w0, gamma0, gamma
5, d, k, x, w0, gamma0, gamma
6, d, n, w0, gamma0, gamma
7, d, x, w_plasma, gamma_plasma, f
8, d, x, w_plasma, gamma_plasma, latticeScalingFactor, N  (тут d = 2 * R_particle)
9, d, x, latticeScalingFactor, N                          (тут d = 2 * R_particle)

Параметры:
d                    - толщина, нм
n                    - комплексный показатель преломления в форме (a; b)
k                    - мнимая часть n
x                    - концентрация AlAs, [0; 1]
w0                   - резонсная частота экситона, эВ
gamma0               - радиационное затухание экситона, эВ
gamma                - нерадиационное затухание экситона, эВ
w_plasma             - плазменная частота для металлических наночастиц в вакууме, эВ
gamma_plasma         - константа затухания плазмона, эВ. gamma_plasma = (h_planck/2Pi) / tau, tau ~ 3E-15 сек, h_planck = 6.582E-16 эВ * сек
f                    - объемная доля нановключений в полупроводниковой матрице, [0; 1]
R_particle           - радиус НЧ металлической фазы (AsSb)
latticeScalingFactor - масштабирующий коэффициент, связывающий эффективный период квадратной решетки НЧ с их радиусом)
N                    - объемная плотность НЧ в полупроводнике, см. формулу Клаузиуса — Моссотти

Пример:
x1
2, 4.5, 0, 0
2, 97, 0.0003, 0.32
x24
5, 11, 0.068, 0.32, 1.4, -0.00127, 0.055
2, 100.77, 0.0004, 0.32
// 7, 1000, 0.32, 7.3, 0.2, 0.0017 - комментарий
